Fields are represented by key words written in UPPER case enclosed inside two curly braces.

When processed by the server, these Fields will be replaced with the data that they represents.

You can click on the "Fields" button to see all available key words.


       Example: 


       {QUANTITY}, {COMPANY-NAME}, {STORE-ADDRESS}


Fields that are colored green represents ROW FIELDS.

A Row Field is linked to each Line Item in the Sales Order.

All Row Fields MUST be enclosed within a pair of ROW TAGS.

Row Tags are represented by  {!ROW-BEGIN}  and  {!ROW-END}


       Example:

     

       <div>

              {!ROW-BEGIN}

                     <b> {QUANTITY} </b>

                     <span> {ITEM-NAME} </span>

              {!ROW-END}

       </div>



The Company Logo is represented by {COMPANY-LOGO} which provides a Base 64 string value of the 

Image. To use this, the key word should be inserted inside the the "src" attribute of an HTML tag.


       Example:


       <img src="data:image/png;base64,{COMPANY-LOGO}">
